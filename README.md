# Thita Lanches - Ribeirão Preto

Sistema de Gerenciamento de Pedidos Eficiente.

## Tecnologias Utilizadas

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como rodar o projeto localmente

1. Clone o repositório
2. Instale as dependências:
   ```sh
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

## Deploy

O projeto pode ser buildado para produção usando:
```sh
npm run build
```
O conteúdo da pasta `dist` pode então ser servido por qualquer servidor web estático.
